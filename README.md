# README #

Node.js library for interfacing with the 20x4 Lcd display for the CTT SensorStation.

### Install ###

```
npm install 
```
### Technologies ###
This project is created with:
- Node.js v10.15.2
- Raspbian GNU/Linux 10 (buster)
